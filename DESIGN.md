# DESIGN.md

## サイト概要

| 項目 | 内容 |
|------|------|
| サイト名 | irodoor |
| 説明文 | 法人イロドアのビジネスを紹介するサイトです。 |
| 対象 | イロドアのビジネスに関心のある方 |

## デザインの方向性

- **テーマ**: かっこいい・洗練された・プロフェッショナル
- **カラー**: ダーク背景（#0a0a0a）、白テキスト、アクセントカラー（#7c6fff）
- **タイポグラフィ**: 大きく力強い見出し、余白を広めに取る
- **レイアウト**: フルワイド、セクションごとに区切る、コンテンツ幅 max 960px

## 作成するページ・セクション

### トップページ（index.astro）
1. **Hero** — 社名・キャッチコピー・サブテキスト
2. **About** — イロドアについての紹介
3. **Business** — 事業内容（カード形式）
4. **Contact CTA** — 問い合わせへの誘導

## 実装メモ

- Header・Footer はベースをそのまま使い、ダークテーマに合わせて CSS でオーバーライドする
- グローバル CSS は変更せず、index.astro の `<style>` 内で `:global()` を使って上書きする
- ブログ機能はそのまま残す（/blog へのリンクは Header に残す）
- SITE_URL・CLOUDFLARE_WEB_ANALYTICS_TOKEN は公開時スキルで設定する
